# Sistema de Gestión de Egresados - Universidad CENFOTEC

## Descripción
Proyecto académico para el curso "Diseño y Programación Web" que implementa un **Sistema de Gestión de Egresados** para la Universidad CENFOTEC. La plataforma permite mantener la conexión entre la universidad y su comunidad de profesionales graduados.

**Autor:** Joao Salas Ramírez  
**Institución:** Universidad CENFOTEC  
**Periodo:** 2026

## Páginas HTML

### 1. **pagina_inicio.html** - Página Principal
- **Descripción:** Página de inicio del sistema autenticado


### 2. **login.html** - Autenticación
- **Descripción:** Página de inicio de sesión

### 3. **perfil_egresado.html** - Perfil del Egresado
- **Descripción:** Perfil personal y actualización de datos 


### 4. **gestion_egresado.html** - Registro de Egresados
- **Descripción:** Formulario administrativo para registrar nuevos egresados
- **Nota:** Para uso del personal del Departamento de Registro

### 5. **gestion_carreras.html** - Gestión de Carreras
- **Descripción:** Formulario administrativo para crear carreras académicas
- **Nota:** Interfaz simple para creación de carreras
